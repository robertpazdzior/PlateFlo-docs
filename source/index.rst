PlateFlo - Plate Scale Perfusion System
=======================================
A modular system for programmable, plate-scale perfusion of adherent cell lines.

File Repository: `OSF (Open Science Foundation)
<https://doi.org/10.17605/OSF.IO/PYJCH>`_

---------------------------------------

This documentation is written to accompany `PlateFlo – A software-controllable
plate-scale perfusion system for culture of adherent cells
<https://doi.org/10.1016/j.ohx.2021.e00222>`_, published in `HardwareX
<https://www.sciencedirect.com/journal/hardwarex>`_ (Oct 2021). See the
peer-reviewed article for a detailed description and contextualization of the
hardware.

.. figure:: hardware/img/graphical_abstract_wPlate.png
   
   **Left** Schematic overview of a multi-plate multi-reservoir perfusion setup
   controlled programmatically using the PlateFlo python package.
   **Right** PlateFlo perfusion plate. A modified Nunc® OmniTray® culture
   plate.
